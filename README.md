# <Nagano_Cake>

## サイト概要
長野県にあるケーキ屋で最近県外からの注文が多くなり、顧客管理システムを導入したい



### サイトテーマ

jqueryを用いてサイトに動きをつけることで、視覚的におしゃれでモダンな雰囲気を表現した。


### テーマを選んだ理由
SNS人気がある店なので、その雰囲気を崩さないようにレイアウト設定した。


### ターゲットユーザ
県外の老若男女

### 主な利用シーン
イベント</br>
祝い事</br>
プレゼント
